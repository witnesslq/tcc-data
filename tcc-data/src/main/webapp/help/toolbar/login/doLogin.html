登陆 – doLogin

接口作用说明

登陆座席到呼叫中心系统，即座席上线

调用方式

doLogin(enterpriseId, hotline, cno, pwd, bindTel, bindType, initStatus, sipIp,sipPwd)

回调函数

cbLogin()

相关参数

参数 要求 描述 备注
enterpriseId 可选 企业ID
hotline 必选 企业热线号码 hotline必须传
cno 必选 座席号 比如2000此座席必须在线
pwd 必选 座席密码 此处密码输入明文，执行过程中，系统会自动md5加密然后传输
bindTel 必选 绑定电话
bindType 必选 绑定电话类型 1：手机或固话，2：分机，3：软电话
initStatus 必选 座席初始状态 online：置闲，pause：置忙，noCase：断线重连
sipIp 可选 软电话IP
sipPwd 可选 软电话密码

返回值

无

示例

var object = {};
object.hotLine = "4006006001";
object.cno = "2000";
object.pwd = "123456"
object.bindTel = "01041005555"
object.bindType = "1";
object.initStatus = "online";
executeAction ('doLogin', object);
回调 - cbLogin()
{'type':'response','code':'0','msg':'ok','reqType':'login','sessionId':'8c291c521d343fa7b94c22bc9b8f6a72','hotline':'4006006001','enterpriseId':'3000100','cno':'2000','cname':'2000','bindTel':'01041005555','bindType':'1','isAgentDebug':'0'}

返回json参数描述

参数名 类型 描述 备注
code String 返回代码 0：登录成功
4：座席不在任何一个队列
29：在线座席数超过并发限制
23：默认自定义置忙状态配置错误
msg String 返回信息
hotline String 热线号码
enterpriseId String 企业号
cno String 座席工号
cname String 座席姓名
bindTel String 绑定电话
bindType String 绑定电话类型
