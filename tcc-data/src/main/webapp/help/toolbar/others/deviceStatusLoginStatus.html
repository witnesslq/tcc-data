过滤状态 – deviceStatusLoginStatus

接口作用说明

过滤当前状态，转换成中文

调用方式

deviceStatus.deviceStatusLoginStatus()

回调函数

无

相关参数

参数 要求 描述 备注
deviceStatus 必选 设备状态
loginStatus 必选 座席状态
pauseDescription 必选 置忙状态描述
busyDescription 必选 通话状态描述

返回值

string(中文)

示例

var str = deviceStatus. deviceStatusLoginStatus(JSON.deviceStatus+JSON.loginStatus, JSON.pauseDescription, JSON.busyDescription);
JSON中必须要包含deviceStatus和loginStatus，pauseDescription和busyDescription可以没有，但是必须要传。
