在线聊天 – doWebChat

接口作用说明

座席聊天

调用方式

doWebChat(JSON)

回调函数

cbWebChat()

相关参数

参数 要求 描述 备注
webchatMsg 必选 发送的信息
webchatType 必选 发送信息的类型 1：发送到指定的座席，2：发送到群
groupId 必选 发送到群的ID 如果发送到座席，groupId为空
cno 必选 座席工号 如果webchatType=1，cno是发送指定座席工号，webchatType=2，cno是发送方座席工号

返回值

无

示例

var object = {};
object.webchatMsg = "发送指定座席2000";
object.webchatType = "1";
object.groupId = "";
object.cno = "2000";
executeAction('doWebChat', object);
回调 - cbWebChat()
返回JSON对象
群聊返回：
{"type":"event","name":"webchatMsg","webchatSenderCno":"2002","webchatSenderCname":"座席2002", "enterpriseId":"3000000","cno":"2002","webchatMsg":"接收信息","webchatType":"2","groupId":"2"}
单聊返回：
{"type":"event","name":"webchatMsg","webchatTargetCno":"2000","webchatTargetCname":"座席2000", "enterpriseId":"3000000","cno":"2002","webchatMsg":"接收信息","webchatType":"1"}

群聊返回json参数描述

参数名 类型 描述 备注
webchatSenderCno String 发送座席号
webchatSenderCname String 发送座席名称
webchatMsg String 接收信息
groupId String 群ID
webchatType String 发送类型 1：发送到单人，2：发送到群
cno String 发送人座席号

单聊返回json参数描述

参数名 类型 描述 备注
webchatTargetCno String 接收座席工号
webchatTargetCname String 接收座席名称
webchatMsg String 接收信息
webchatType String 发送类型 1：发送到单人，2：发送到群
cno String 发送人座席号
