转移IVR – doInteract

接口作用说明

座席保持（转移以后座席听等待音乐）

发起者

第三方

接收者

呼叫中心

调用方式

doInteract(JSON)

回调函数

cbInteract()

相关参数

参数名 要求 描述 备注
interactObject 必选 转移值 格式：ivr_id,ivr_node
objectType 必选 转移类型 0：普通电话，1：座席号，2：分机，3：IVR需要从IVR接口调出IVR节点

返回值

无

示例

var object = {};
object.interactObject = "2,1.1";
object.objectType = "3";
executeAction('doInteract', object);
回调 - cbInteract()
返回的JSON对象
{"type":"response","code":"0","msg":"ok","reqType":"interact"}

返回json参数描述

参数名 类型 描述 备注
code String 返回代码 0：成功，2：连接已中断，10：转移参数错误
msg String 返回信息
