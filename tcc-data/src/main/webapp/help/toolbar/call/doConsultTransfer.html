咨询转接 – doConsultTransfer

接口作用说明

咨询转接

发起者

第三方

接收者

呼叫中心

调用方式

doConsultTransfer(JSON)

回调函数

cbConsultTransfer()

相关参数

参数名 要求 描述 备注
limitTimeSecond 可选 咨询转移后最长通话时间 600秒=10分钟
limitTimeAlertSecond 可选 距离通话结束前提醒时间 60秒，提前一分钟提醒
limitTimeFile 可选 提醒文件名 如果为空，默认语音为“您好，此次通话还有1分钟结束”
limitTimeSecond应该大于limitTimeAlertSecond，如果limitTimeAlertSecond如果为0表示不启用提醒功能。
limitTimeFile格式；
例如：公共语音库on_duty；企业语音库：3000072/xxxxxxx
名称不带.wav后缀

返回值

无

示例

var object = {};
object.limitTimeSecond = "600";
object.limitTimeAlertSecond = "60";
object.limitTimeFile = "xxx";
executeAction('doConsultTransfer', object);
回调 - cbConsultTransfer()
返回的JSON对象
{"type":"response","code":"0","msg":"ok","reqType":"consultTransfer"}

返回json参数描述

参数名 类型 描述 备注
code String 返回代码 0：成功，2：连接已中断，5：没有正确拿到此座席咨询转接的channel
msg String 返回信息
