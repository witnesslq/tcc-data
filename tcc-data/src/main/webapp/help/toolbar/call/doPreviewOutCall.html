外呼 – doPreviewOutCall

接口作用说明

外呼

发起者

第三方

接收者

呼叫中心

调用方式

doPreviewOutCall(JSON)

回调函数

cbPreviewOutCall()

相关参数

参数名 要求 描述 备注
tel 必选 电话号码
callType 必选 外呼电话类型 1：呼入，2：网上400,3：点击外呼，4：预览外呼，5：IVR外呼，6：直接外呼
taskId 可选 预览式外呼任务ID
customerCrmId 可选 CrmID
taskInventoryId 可选 预览式外呼详细ID
userField 可选 用户自定义参数 String类型
obClidLeftNumber 可选 外显号码 String类型

返回值

无

示例

var object={};
object.tel = "01041005555";
object.callType = '3';
object.taskId = "1";
object.customerCrmId="456";
object.taskInventoryId = "75874";
object.userField = “test”;
object.obClidLeftNumber = “80000816”;
executeAction('doPreviewOutCall', object);
回调 - cbPreviewOutCall()
返回的JSON对象
{"type":"response","code":"0","msg":"ok","reqType":"previewOutCall"}

返回json参数描述

参数名 类型 描述 备注
msg String 返回信息
