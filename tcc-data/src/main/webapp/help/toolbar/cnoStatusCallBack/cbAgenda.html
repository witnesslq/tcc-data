预约提醒 – cbAgenda

接口作用说明

预约提醒回调

调用方式

日程表预约到预定时间

回调函数

cbAgenda()
返回JSON对象：
{"type":"event","name":"agenda","enterpriseId":"3000000","cno":"2002", "agendType":"1","agenda_subject":"主题","agenda_customer_name":"客户姓名", "agenda_customer_phone_number":"15810716666","agenda_comment":"备注", "agenda_bookingTime":"2012-12-03 15:04:00.0"}

返回json参数描述

参数名 类型 描述 备注
enterpriseId String 企业ID
cno String 座席号
agendType String 预约类型 1：联系客户，2：会议，3：任务，4：其他
agenda_subject String 主题
agenda_customer_name String 联系人姓名
agenda_customer_phone_number String 联系人电话
agenda_comment String 备注
agenda_bookingTime String 预约提醒时间
