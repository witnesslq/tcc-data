未接来电 – cbUnanswer

接口作用说明

未接来电回调

调用方式

未接听电话时候调用

回调函数

cbUnanswer()
返回JSON对象：
{"type":"event","name":"unanswer","enterpriseId":"3000000","callType":"1", "customerNumber":"01087125968","customerNumberType":"1","customerAreaCode":"010", "startTime":"1354516837","qno":"30000000001"}

返回json参数描述

参数名 类型 描述 备注
startTime String 开始时间
customerNumber String 电话号码
enterpriseId String 企业号
callType String 电话类型 0：监听通话，1：呼入，2：网上400，3：点击外呼，4：预览外呼，5：IVR外呼，6：直接外呼
customerAreaCode String 号码归属地
