获取座席状态 – doStatus

接口作用说明

获取座席状态

调用方式

doStatus()

回调函数

cbStatus()

相关参数

无

返回值

无

示例

executeAction('doStatus');
回调 - cbStatus()
{"type":"event","name":"status","enterpriseId":"3000000","cno":"2002","deviceStatus":"idle","loginStatus":"online"}

返回json参数描述

参数名 类型 描述 备注
customerAreaCode String 号码归属地
customerNumberType String 号码类型 1：固话，2：手机
callType String 电话类型 1：呼入，2：网上400，3：点击外呼
4：预览外呼，5：IVR外呼 6：直接外呼
enterpriseId String 企业号
cno String 座席工号
customerNumber String 电话号码
deviceStatus String 设备状态 1：idele，2：ringing，3：busy
loginStatus String 座席状态 1：online，2：pause
pauseDescription String 置忙描述 pauseWrapup：整理
busyDescription String 通话描述 busyOnhold：保持
