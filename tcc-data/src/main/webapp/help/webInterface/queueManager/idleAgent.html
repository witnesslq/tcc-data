获取企业在线空闲座席清单

接口作用说明

通过接口获取企业当前空闲座席清单

API URL

接口URL（POST方式）
发起者：合作方
接受者：天润融通
接口类型：HTTP接口
调用方式：POST
返回格式：JSON数据
URL规范：
http:// InterfaceServerIp:port/interface/entrance/OpenInterfaceEntrance?interfaceType=freeClients &enterpriseId=3000000&userName=admin&pwd=c33367701511b4f6020ec61ded352059（注意：interfaceType参数值前后不要有空格）
说明：InterfaceServerIp:port 为使用账号登录www.ccic2.com之后，在浏览器地址栏看到的域名地址。
例如：xa.ccic2.com
           bj.ccic2.com
           1.ccic2.com

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业Id 必选企业id
interfaceType 必选 请求类型 2: 获取企业当前的空闲坐席清单
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 对应username的密码，md5(登录密码)，例如：md5(123456)
