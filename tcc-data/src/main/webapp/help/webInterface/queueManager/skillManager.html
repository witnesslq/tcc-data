技能接口

接口作用说明

通过接口添加技能、修改技能、删除技能、查看企业所有技能

新建或修改技能

接口类型：HTTP
接口使用账户：企业账户
接口地址：呼叫中心系统IP
接口参数：企业账号、企业账户密码、技能id、名称、备注
接口返回值：操作结果
接口说明
添加或修改一个技能。
接口URL
URL示例： http://172.16.203.197/interfaceAction/skillInterface!save.action?enterpriseId=3000008&userName=admin&password=e10adc3949ba59abbe56e057f20f883e&name=%E5%8C%97%E4%BA%AC&comment=%E5%8C%97%E4%BA%AC
说明：该示例中password = e10adc3949ba59abbe56e057f20f883e = MD5Encoder.encode('123456')。name值是经过URLEncode按“UTF-8”格式编码的，中文是“北京”。
参数说明
参数 要求 描述 备注
username 必选 角色名称 例如:admin
password 必选 登录密码 md5(登录密码))
id 可选 技能id 技能id,如新增不用传.
name 必选 技能名称 需进行URLEncode编码
comment 必选 描述 需进行URLEncode编码
 
返回值
JSON串，格式为{result:xxx,msg:xxx},result表示返回结果状态码，msg表示结果状态的描述。
如果新增技能成功,将返回技能Id，JSON串格式为{result:xxx,msg:xxx,id:xxx}
result与msg属性值的定义：
result msg
success 保存成功
error 失败原因

删除技能

接口类型：HTTP
接口使用账户：企业账户
接口地址：呼叫中心系统IP
接口参数：企业账号、企业账户密码、技能Id
接口返回值：操作结果
接口说明
删除一个指定技能。
接口URL
URL示例： http://172.16.203.197/interfaceAction/skillInterface!delete.action?enterpriseId=3000008&userName=admin&password=e10adc3949ba59abbe56e057f20f883e&id=6
说明：该示例中password = e10adc3949ba59abbe56e057f20f883e = MD5Encoder.encode('123456')。
参数说明
参数 要求 描述 备注
username 必选 角色名称 例如:admin
password 必选 登录密码 md5(登录密码))
id 必选 技能编号 技能id
返回值
JSON串，格式为{result:xxx,msg:xxx},result表示返回结果状态码，msg表示结果状态的描述。
 
result与msg属性值的定义：
result description
success 删除成功
error 失败原因

查看所有技能

接口说明
查看某企业下的所有技能
接口URL
URL示例： http://172.16.203.197/interfaceAction/skillInterface!list.action?enterpriseId=3000008&userName=admin&password=e10adc3949ba59abbe56e057f20f883e
说明：该示例中password = e10adc3949ba59abbe56e057f20f883e = MD5Encoder.encode('123456')。
参数说明
参数 要求 描述 备注
username 必选 角色名称 例如:admin
password 必选 登录密码 md5(登录密码))
返回值
JSON串，格式为{"result":"XXX","msg":{"data":[{},{}…]}},result表示返回结果状态，msg表示包含所有技能基本信息的JSON格式数据的数组
result、msg与description的定义：
result msg description count
success “data":[{"id":1,"name":"咨询","comment":"咨询技能"},{"id":2,"name":"投诉","comment":"投诉技能"}] 成功返回各技能信息 data数组的长度
error [] 失败原因 0
 
data属性值的定义：
属性名 描述 数据长度或格式 取值范围 例子
id 技能编号      
name 名称     咨询
comment 备注     咨询技能

验证技能名称是否重复

接口类型：HTTP
接口使用账户：企业账户
接口地址：呼叫中心系统IP
接口参数：企业账号、技能名称、技能Id
接口返回值：操作结果
接口说明
验证技能名称是否重复。
接口URL
URL示例： http://172.16.203.197/interfaceAction/skillInterface!validateName.action?enterpriseId=3000008&name=%E6%8A%95%E8%AF%89&id=
 
参数说明
参数 要求 描述 备注
enterpriseId 必选 企业Id 例如:3000008
name 必选 技能名称 URLEncode编码表示
id 可选 技能编号 技能id,如为新增则不用传
返回值
JSON串，格式为{"result":"XXX"},result表示返回验证结果。
 
result的定义：
result description
true 已存在
false 未存在
