座席管理接口

接口作用说明

通过接口获取企业当前空闲座席清单

概述

企业自己有CRM业务系统，业务系统为B/S或C/S架构，需在通过HTTP接口对座席进行管理。
规范
TIWebServer：天润呼叫中心接口服务器IP或域名
TIPort：天润呼叫中心接口服务器端口
接口URL中的TIWebServer: TIPort地址，是登录座席前台或管理员后台在浏览器地址栏看到的域名地址，如图标注部分：

WebServer：企业接口服务器IP或域名
Port：企业接口服务器端口

座席上线

接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，座席工号，上线初始状态，绑定电话，电话类型
接口返回值：JSON格式返回结果
接口URL
URL示例：
http://TIWebServer:TIPort/interface/client/ClientOnline?enterpriseId=&userName=&pwd=&seed=&cno=&status=&bindTel=&type=
参数说明
参数 要求 描述 备注
enterpriseId 必选 企业编号 管理员后台登陆后，位于右上角位置
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 需和seed进行MD5加密，格式：md5(md5(对应userName的密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串  
cno 必选 座席工号 上线的座席工号
status 必选 上线初始状态 1. 空闲  2. 置忙
bindTel 必选 绑定电话  
type 必选 电话类型 1. 普通电话 2. 分机
返回值
JSON格式数据：
{result:返回结果, code:结果代码,msg:描述}
 
result code msg
success 0 操作成功
error 1 参数[具体参数名]不能为空
error 2 参数[具体参数名]格式不正确
error 3 企业不存在或状态不正常
error 4 无权操作，接口未开通
error 5 验证失败，用户名或密码不正确
error 6 座席未激活，不能登录电话
error 7 绑定电话格式不正确
error 8 绑定电话已经被使用
error 9 分机号码不存在
error 10 绑定电话无对应路由
error 11 在线座席数超过并发限制
error 12 座席已经登录
error 13 座席正在通话中，不能切换电话登录
error 14 座席不在任何一个队列，请检查座席配置
error 99 其他错误，请联系管理员

座席状态推送

接口类型：HTTP请求
推送方式：POST
接口地址：企业系统推送信息接收地址
接口返回值：JSON格式返回结果
接口URL
URL示例：
http://WebServer:Port/interface?
参数变量说明
参数 描述 备注
enterpriseId 企业编号 管理员后台登陆后，位于右上角位置
cno 座席工号 座席工号
loginStatus 座席状态 取值为：online：在线、offline：离线、pause：置忙
deviceStatus 设备状态 取值为：idle：空闲、ringing：响铃、busy：忙碌
pauseDescription 置忙描述 loginStatus=pause时有值，取值为：座席前端设置中座席端自定义置忙状态中的状态名称、pauseWrapup（整理）、pauseOutCalling（外呼中）
wrapupTime 整理时间 pauseDescription=pauseWrapup时有值，秒数，表示整理的时长
busyDescription 通话描述 取值为：busyOnhold
customerNumber 主叫号码 deviceStatus=busy时有值，来电呼入或外呼的客户号码
customerNumberType 号码类型 deviceStatus=busy时有值，1：固话、2：手机
callType 呼叫类型 deviceStatus=busy时有值，1：呼入、2：网上400、3：点击外呼、4：预览式外呼、5：预测式外呼、6：摘机外呼
consulterCno 咨询自座席号 deviceStatus=busy且事件来自咨询时，此参数为咨询来自的座席工号
transferCno 转移自座席号 deviceStatus=busy且事件来自转移时，此参数为转移来自的座席工号
座席状态判断示例
状态 loginStatus deviceStatus pauseDescription busyDescription
空闲 online idle    
置忙 pause idle 小休/忙碌/WC…  
响铃（呼入） online ringing    
响铃（外呼） pause ringing pauseOutCalling/小休/忙碌/WC…  
通话（呼入/空闲外呼） online busy    
通话（置忙外呼） pause busy pauseOutCalling/小休/忙碌/WC…  
保持 pause busy   busyOnhold
整理 pause idle pauseWrapup  
外呼中 pause idle pauseOutCalling  
离线 offline idle

咨询

接口参数：企业编号，用户名，密码，座席工号，咨询类型，咨询对象
接口返回值：JSON格式返回结果
 
接口URL

http://TIWebServer:TIPort/interface/Consult?enterpriseId=&userName=&seed=&pwd=&cno=&consultObject=&objectType=
参数说明

0：电话号码
1：座席工号
2：分机号
consultObject 必选 咨询对象 objectType=0时为电话号码 
objectType=1时为座席工号

挂断

接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，座席工号
接口返回值：JSON格式返回结果
接口URL
URL示例：
http://TIWebServer:TIPort/interface/HangUp?enterpriseId=&userName=&seed=&pwd=&cno=
参数说明
参数 要求 描述 备注
enterpriseId 必选 企业编号 管理员后台登陆后，位于右上角位置
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 需和seed进行MD5加密，格式：md5(md5(对应userName的密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串  
cno 必选 座席号 比如2000，此座席必须在通话中
 
返回值
JSON格式数据
{result:返回结果, code:结果代码,msg:描述}
result code msg
success 0 座席挂断成功
error 1 参数[具体参数名]不能为空
error 2 参数[具体参数名]格式不正确
error 3 传入的企业id或者热线号码有误
error 4 无权操作，接口未开通
error 5 密码错误
error 6 传入的座席工号有误
error 7 座席[具体座席号]不在线
error 8 座席[具体座席号]不在通话中
error 9 出现异常

http://172.16.203.182/interface/HangUp?enterpriseId=3010066&userName=admin&seed=1234&pwd=b605e86d02eef8bfd0646f6a704c17c9&cno=2001
