大屏幕监控接口

接口作用说明

实时监控队列中各状态的值

API URL

接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码
接口返回值：JSON数据或展示界面
接口URL
URL示例：
http://172.16.203.210/QueueStaticByName?enterpriseId=4000000&queue=40000000000&refreshtime=5

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业编号 管理员后台登陆后，位于右上角位置
queue 可选 队列号(此队列号=企业id+队列号) 当不输入queue,是按所有队列查找
refreshtime 必选 刷新时间 最少不能小于5秒
返回值：
如果参数正确返回结果是展示界面

参数不正确则返回json错误信息(例)
{"error":"无此队列"}
