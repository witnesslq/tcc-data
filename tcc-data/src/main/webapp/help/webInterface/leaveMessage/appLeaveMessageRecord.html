留言记录接口

接口作用说明

查看留言记录。

新建或修改接口

接口URL（GET方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：呼叫中心系统IP
接口参数：企业账号、企业账户密码、技能id、名称、备注
接口返回值：操作结果
接口URL
URL示例：
http://172.16.203.190/interfaceAction/commentRecordInterface!view.action?enterpriseId=3000002&userName=admin&pwd=e10adc3949ba59abbe56e057f20f883e&customerNumber=15120054657&hotline=4006004321&startTime=2013-09-01%2000:00:00&endTime=2013-09-30%2014:39:49

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业id enterpriseId与hotline必须二选一
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 对应username的密码，md5(登录密码），例如：md5(123456)
customerNumber 可选 来电号码 添加来电号码查询条件
hotline 可选 热线号码 添加热线号码查询条件
startTime 可选 开始时间 格式“yyyy-mm-dd hh:mm:ss”，例如：2012-08-01 00:00:00，默认当月第一天
endTime 可选 结束时间 格式“yyyy-mm-dd hh:mm:ss”，例如：2012-08-08 00:00:00，默认当月最后一天
start 可选 从第几条开始取 正整数，大于等于0，默认0
limit 可选 需要取出的数据条数 正整数，大于0，不输入默认1000条

返回值

JSON串，格式为{result:xxx,msg:xxx},result表示返回结果状态码，msg表示结果状态的描述。
JSON格式数据
{"result":"success","msg":{"data":[{"uniqueId":"wangjing- 1378187478.0","customerNumber":"15120054657","customerProvince":"北 京","customerCity":"北 京","clientNumber":"","hotline":"4006004321","startTime":"2013-09-03 13:51:18","endTime":"2013-09-03 13:51:47","bridgeDuration":"0","totalDuration":"29"}],"paging":{"start":0,"totalSize":1,"pageSize":1}}}
名称 描述 数据格式 取值范围 例子
uniqueId 电话唯一标识 ccic_dev_11-1334563616.70
customerNumber 来电号码 区号 + 7或8位的固话号码，区号与固话号码之间无'-'；或11位长度的手机号码。 中国国内固话或手机 01059222999或18701051984
customerProvince 来电省份 北京
customerCity 来电城市 北京
startTime 电话进入系统时间 时间格式“yyyy-mm-dd hh:mm:ss” 2012-05-01 15:00:02
endTime 结束时间 时间格式“yyyy-mm-dd hh:mm:ss” 2012-05-01 15:00:02
bridgeDuration 通话时长 格式“hh:mm:ss” 00:02:26
totalDuration 总时长 格式“hh:mm:ss” 00:02:36
res返回值 msg
success 操作成功
error 失败原因
URL示例：
使用enterpriseId
http://xxx.ccic2.com/interface/ivrCallback?enterpriseId=3000072&userName=admin&pwd=66b7146abd2f6fc7d78e854f0c94cffb&uniqueId=dev18-1415787391.4&paramNames=a&a=1
成功返回：
{"result":"success","code":"0","msg":""}
失败返回：
