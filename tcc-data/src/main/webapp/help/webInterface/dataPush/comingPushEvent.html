来电事件推送

API URL

接口URL（GET方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业账号、来电号码、工号、呼叫类型、callid（呼叫唯一标识）、队列号、被叫号码
返回值：网页


企业需要把对接的接收信息的URL在呼叫中心系统中进行设置。
进入“系统设置”—“接口设置”—“CRM对接设置”


先填写推送接口的URL，方式选择“来电推送”，再配置对应可选参数及自定义参数名称;
如：crmId参数在您的系统中为userid参数，那么在crmId参数下面的对应输入框填写userid;



接口URL URL示例
http://WebServer:Port/ringing.jsp?enterpriseId=3000028&hotline=4006604889&cno=2000&tel=01087120766&customerNumberType=1&customerAreaCode=010&calltype=1&numberTrunk=63352451&uniqueId=xian001-1363743441.417&taskInventoryId=undefined&taskId=undefined&qid=30000280011&consultCon=&transferCno=

参数说明

参数 描述 备注
enterpriseId 企业id 管理员后台登陆后，位于右上角位置
hotline 企业热线号码 企业登陆使用的热线号码，包括：400号码或固话
bindTel 绑定电话
cno 坐席工号
customerNumber 主叫号码 来电呼入号码或者外呼客户号码
customerNumberType 来电或外号码类型 手机或者固话
customerAreaCode 号码地区 呼入电话或者外呼客户电话所在地区
calltype 呼叫类型 1：呼入、2：网上400、3：点击外呼、4：预览式外呼、5：预测式外呼、6：摘机外呼
numberTrunk 目的码 系统号码
uniqueId 呼叫唯一标识 即callid，用于接口取得通话记录和录音等
taskId 外呼任务id 外呼任务的标识
taskInventoryId 外呼任务详情ID 外呼任务的内部标识
qid 呼入队列号 企业编号+四位队列号
consulterCno 咨询自座席号 事件来自咨询时，此参数为咨询来自的座席工号
transferCno 转移自座席号 事件来自转移时，此参数为转移来自的座席工号
自定义参数变量 ivr变量 在IVR语音导航使用的变量可以传递到来电推送信息中

返回值

{'reuslt':'success'}
