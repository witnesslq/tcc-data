外呼IVR记录详细

实现步骤

接口URL（POST方式）
发起者：合作方
接受者：天润融通
接口类型：HTTP接口
调用方式：POST
返回格式：JSON数据
URL规范：
http://InterfaceServerIp:port /interfaceAction/cdrObInterface!listCdrObIvrDetail.action
说明：InterfaceServerIp:port 为使用账号登录www.ccic2.com之后，在浏览器地址栏看到的域名地址。
例如：xa.ccic2.com
           bj.ccic2.com
           1.ccic2.com

相关参数

参数 要求 描述 备注
hotline 可选 企业热线号码 与enterpriseId二选一
enterpriseId 可选 企业Id 与hotline二选一
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 对应username的密码，md5(md5(登录密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串  
uniqueId 必选 通话记录uniqueId 对应通话记录的uniqueId

返回值

此接口返回JSON格式数据。
JSON格式数据
{result:返回结果,
msg:{
data:[{通话记录IVR详细数据},{}..]
}
}
 
result msg description
success {“result”:”success”,”msg”:{“data”: [{“ivrName”:”转咨询队列”,”path”:”1”,”athname”:”欢迎”,”action”:”队列”,”startTime”:”2012-08-09 14:29:14”,”endTime”:”2012-08-09 14:30:14”}….]
}}		 成功返回通话记录IVR详细信息
error “无权访问” 查询通话记录IVR详细失败原因
说明： data取值为json数组。
 
json格式数据记录data的各属性定义：
名称 描述 数据格式 取值范围 例子
ivrName 语音导航名称     转咨询队列
path 节点     1
pathName 节点名称     欢迎
action 节点类型     队列
startTime 开始呼叫时间     2012-08-09 14:29:14
endTime 结束时间     2012-08-09 14:30:14
pressKey 按键值（只有节点类型为选择或收号时有此数据）     1
pressTime 按键时间（只有节点类型为选择或收号时有此数据）     2012-08-09 14:30:12
