对接方式介绍

对接说明

实现通话记录内容及录音的查询、数据本地化存储（录音下载、在线试听）、Excel导出、推送等功能

对接方式

对接方式有6种：
①：外呼通话记录
介绍
查询外呼通话记录的主体，可根据时间段、接听状态等进行查询，同时可实现录音的视听及下载，内容与前后台外呼通话记录显示一致
②：外呼通话记录详细
介绍
查询一通外呼通话记录的详细内容，须根据通话记录唯一ID查询，内容与前后台外呼通话记录显示一致
③：外呼通话记录DB同步
介绍
直接从数据库取外呼通话记录主体，数据为原始数据
④：外呼通话记录明细DB同步
介绍
直接从数据库取外呼通话记录明细，数据为原始数据
⑤：外呼IVR记录详细
介绍
获取外呼通话记录进入的IVR详情，数据与前后台通话记录中IVR详情相同（外呼转移IVR有此数据，通常没有数据）
⑥：外呼通话记录根据查询条件导出Excel接口
