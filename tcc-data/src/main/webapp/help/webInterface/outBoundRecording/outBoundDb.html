外呼通话记录DB同步

实现步骤

接口URL（POST方式）
发起者：合作方
接受者：天润融通
接口类型：HTTP接口
调用方式：POST
返回格式：JSON数据
URL规范：
http://InterfaceServerIp:port/interfaceAction/cdrObInterface!cdrObDBCopy.action
说明：InterfaceServerIp:port 为使用账号登录www.ccic2.com之后，在浏览器地址栏看到的域名地址。
例如：xa.ccic2.com
           bj.ccic2.com
           1.ccic2.com

相关参数

参数 要求 描述 备注
hotline 可选 企业热线号码 与enterpriseId二选一
enterpriseId 可选 企业Id 与hotline二选一
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 对应username的密码，md5(md5(登录密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串  
id 必选 数据表主键标识 正整数，大于等于0，默认0
limit 可选 通话记录条数 正整数，大于等于1，默认为10，最大不能超过5000
userField 可选 用户自定义参数 任意字符
注：每次请求返回行数最大值为limit（最大不能超过5000条），即从id大于传入的参数id的值开始按id升序取出limit条记录。

返回值

此接口返回JSON格式数据。
JSON格式数据
{result:返回结果,
msg:{
data:[{通话记录数据},{}..],
totalCount:总记录数
}
}

名称（数据库字段名） 描述 数据库格式 取值范围 例子

client_number 座席号码 character varying    
client_area_code 座席区号 character varying    
cno 座席号 character varying    
exten 如果是分机保存分机号 character varying,    
client_name 座席名称 character varying    
client_crm_id 座席crmid character varying    
start_time 客户进入系统时间 bigint 秒数（相对与January 1, 1970, 00:00:00 GMT） 如：1333940942即：2012-04-09 11:09:02
answer_time 系统接听时间 bigint 秒数（相对与January 1, 1970, 00:00:00 GMT）  
bridge_time 座席应答时间 bigint 秒数（相对与January 1, 1970, 00:00:00 GMT）  
end_time 结束时间 bigint 秒数（相对与January 1, 1970, 00:00:00 GMT）  
bill_duration 计费时长 integer DEFAULT 0 正常以秒计算时间 如：84
即：1分24秒
bridge_duration 座席接听时长 integer DEFAULT 0 正常以秒计算时间  
total_duration 总通话时长 integer DEFAULT 0 正常以秒计算时间

ivr_id IVR资源id integer    
ivr_name IVR资源名称 character varying    
queue_name 队列名 character varying    
record_file 录音文件，文件名保存相对路径 character varying   将unique_id后半段的时间戳转化成yyyyMMdd加在查询出的record_file前面用/隔开
score 录音质检打分 0为未打分 integer DEFAULT 0    
score_comment 打分备注 character varying 可能为NULL  
in_case_lib 是否在案例库中0:不在 1:在 integer DEFAULT 0    
task_id 如果在外呼任务中存储外呼任务对应id integer    
task_name 如果在外呼任务中存储外呼任务名 character varying

mark 标记 2:咨询 3:转移 integer DEFAULT 0    
mark_data 标记的数据：咨询的座席，转移的座席。支持多组用逗号分隔 character varying    
end_reason 挂机原因 1000:客户挂机 1001:座席挂机 1002:被强拆 1-127:其他挂机原因 integer DEFAULT 0    
gw_ip 网关ip character varying    
user_field 用户自定义参数 character varying    
combo_cost 外呼套餐扣费 integer DEFAULT 0    
is_billing 是否计费 integer DEFAULT 1    
create_time 记录创建时间 timestamp with time zone DEFAULT now()   2012-04-09 11:09:03
