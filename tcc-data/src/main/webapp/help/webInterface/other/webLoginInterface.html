网上登录接口

表单方式

[表单方式]
在代理商的网页中嵌入表单HTML代码，即可在网页中跳转到自服务系统，无需登录产品网站。
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业账号、企业账户密码、验证码、来源域名、定制域名
返回值：跳转到系统
接口URL
URL示例：
http://ccic.ti-net.com.cn/interface/login.jsp?username=4006667773&password=6752b86e91e26852346bc986fa445765&authCode=pm8b&fromURL=www.call4006.com&toURL=www.smart400.com
参数 要求 描述 备注
username 必选 登录400号码 4001，4006，4007或4008开头的十位数字
pwd 必选 密码 md5(username+password+ authCode)
authCode 必选 验证码 根据验证码接口生成的验证码
fromURL 可选 来源域名 退出或登录失败跳转到的URL，需使用URLEncode编码
toURL 可选 定制域名码 登陆成功跳转到的URL，需使用URLEncode编码

返回值

若登陆成功，跳转到来源域名toURL；若登录失败跳转到fromURL。

Ajax方式

[Ajax方式]
通过Ajax方式调用接口验证用户名及密码，再进行相应后续处理。
如：跳转到自服务系统页面，并同时在自己的系统中作记录
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业账号、企业账户密码、验证码
返回值：操作结果
接口URL
URL示例：
http://ccic.ti-net.com.cn/interface/loginAjax.jsp?username=4006667773&password=6752b86e91e26852346bc986fa445765&authCode=pm8b
参数 要求 描述 备注
username 必选 登录400号码 4001，4006，4007或4008开头的十位数字
pwd 必选 密码 md5(username+password+ authCode)
authCode 必选 验证码 根据验证码接口生成的验证码

返回值

返回登录状态，JSON串，格式为{result:xxxx,description:xxxx},result表示返回结果状态码，description表示结果状态的描述。


result与description属性值的定义：
result description
0 登录失败原因
1 登录成功

验证码生成接口

接口URL
URL示例：
http://ccic.ti-net.com.cn /system!getLoginSecurityCodeImage.action

参数说明


不需要参数

返回值


返回随机的数字与字母混合的验证码图片。
