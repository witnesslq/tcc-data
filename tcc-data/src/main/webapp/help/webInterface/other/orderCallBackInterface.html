一、查询智能预约回呼记录

实现步骤

接口URL（POST方式）
发起者：合作方
接受者：天润融通
接口类型：HTTP接口
调用方式：POST
返回格式：JSON数据
URL规范：
http://InterfaceServerIp:port/interfaceAction/ocbInterface!list.action
说明：InterfaceServerIp:port 为使用账号登录http://InterfaceServerIp:port /interfaceAction/ocbInterface!list.action之后，在浏览器地址栏看到的域名地址。
例如：http://InterfaceServerIp:port/interfaceAction/ocbInterface!list.action

相关参数

参数 要求 描述 备注
hotline 可选 企业热线号码 与enterpriseId二选一
enterpriseId 可选 企业Id 与hotline二选一
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 对应username的密码，md5(md5(登录密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串  
cno 可选 执行此外呼的坐席  
qno 可选 队列号 执行该预约记录的坐席所在的队列 
tel 必选 电话号码 改预约记录的电话
isCall 可选 是否回呼 0表示还未回呼,1表示已经回呼,2表示呼叫中,3表示呼叫失败
orderTime 可选 预约时间 预约回呼的时间,没有提交的话,默认为空串
callBackTime 可选 回呼时间 执行呼叫该预约记录的时间,默认为空串
createTime 可选 创建时间 创建这条预约记录的时间,默认为空串
startTime 可选 开始时间 格式“yyyy-mm-dd hh:mm:ss”，例如：2012-08-01 00:00:00，默认当月第一天，此参数需要URLEncode
endTime 可选 结束时间 格式“yyyy-mm-dd hh:mm:ss”，例如：2012-08-08 00:00:00，默认当月最后一天，此参数需要URLEncode

返回值

此接口返回JSON格式数据。
JSON格式数据
{result:返回结果,
msg:{
}
}
 
result msg description
success {"result":"success","msg":{"data":[{"id":"1","cno":"2000","qno":"0000","tel":"01087120766","isCall":"1","customerProvince":"北京","customerCity":"北京","queueName":"0001","startTime":"2012-08-09 14:29:13","startTime":"2012-08-09 14:29:13","orderTime":"2012-08-09 14:29:13","callBackTime":"2012-08-09 14:29:13","createTime":"2012-08-09 14:29:13",},...]}} 成功返回预约回呼记录
error "无权访问" 查询预约回呼记录失败原因
说明： data取值为json数组。
 
json格式数据记录data的各属性定义：
名称 描述 数据格式 例子
id 预约记录标识    
hotline 热线号码    
tel 来电号码 区号 + 7或8位的固话号码，区号与固话号码之间无'-'；或11位长度的手机号码。中国国内固话或手机 01059222999
或18701051984
customerProvince 来电省份   北京
customerCity 来电城市   北京
queueName 队列号   0001
cno 座席工号   1001
startTime 要查询记录的开始时间 时间格式“yyyy-mm-dd hh:mm:ss” 2012-05-01 15:00:02
aorderTime 预约回呼的时间 时间格式“yyyy-mm-dd hh:mm:ss” 2012-05-01 15:00:12
createTime 创建这条预约记录时间 时间格式“yyyy-mm-dd hh:mm:ss” 2012-05-01 15:00:12
isCallBackTime 回呼这条预约记录时间 时间格式“yyyy-mm-dd hh:mm:ss” 2012-05-01 15:00:12
isCall 是否回呼   1

              <input type="text" name="isCallBackTime" value="" /></br>
              <input type="text" name="createTime" value="" /></br>
              <input type="text" name="tel" value="" /></br>
              <input type="text" name="startTime" value="2012-08-08 00:00:00" /></br>
