平台监控接口

API URL

接口URL（GET方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，随机字符串，任务类型和任务id
返回值：JSON格式返回结果
接口URL
URL示例：
http://InterfaceServerIp:port/interface/serviceStatus.jsp?seed=?id=?

相关参数

参数 要求 描述 备注
id 必选 请求id标识
seed 必选 随机字符串

返回值

JSON格式数据
{"result":"success","id":"1","seed":"a","msg":{"status":1,"describe":"正常","information":"正常"}
名称 描述 取值范围 例子
status 状态 0异常,1正常
describe 描述 文本 呼叫故障
information 详情 文本 电路故障
