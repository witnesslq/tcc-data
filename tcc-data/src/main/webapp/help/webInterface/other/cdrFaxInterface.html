传真记录请求接口

API URL

接口URL（POST方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，随机字符串，任务类型和任务id
返回值：JSON格式返回结果
接口URL
URL示例：
http://InterfaceServerIp:port/interfaceAction/cdrFaxInterface!listCdrFax.action

相关参数

参数 要求 描述 备注
enterpriseId 可选 企业id 与hotline二选一
hotline 可选 企业热线号码 与enterpriseId二选一
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 对应username的密码，md5(md5(登录密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串 随机字符串：例如：aaa
type 必选 传真类型 传真类型 0:全部 1:接收 2:发送
sendNumber 可选 发送方号码 与type形成组合条件，如type=2，sendNumber=01087120766，则表示查询所有发送传真中发送方号码为01087120766的记录
sendNumber 可选 发送方号码 与type形成组合条件，如type=2，sendNumber=01087120766，则表示查询所有发送传真中发送方号码为01087120766的记录
receiveNumber 可选 接收方号码 与type形成组合条件
startTime 必选 开始时间 格式“yyyy-mm-dd hh:mm:ss”，例如：2012-08-01 00:00:00，默认当月第一天
endTime 必选 结束时间 格式“yyyy-mm-dd hh:mm:ss”，例如：2012-08-08 00:00:00，默认当月最后一天
start 必选 从第几条开始取 正整数，大于等于0，默认0
limit 必选 需要取出的数据条数 正整数，大于0，最大不能超过1000，默认10

返回值

返回JSON：
JSON串，格式为
{result:返回结果, msg:{ data:[{传真记录数据},{}..], paging:{"从第几条开始取":"0","总条数":15,"本页条数":10} } }
result msg description
success {"result":"success","msg":{"data":[{"enterpriseId":3000000,"type":1,"startTime":"2012-12-10 00:00:00","sendNumber":"01041005966","receiveNumber":"01041005968","pageCount":1,"file":"http://172.16.203.190/voices/fax_receive/3000000aaa1355194343536.pdf"}],"paging":{"start":0,"totalSize":1,"pageSize":2}}} 成功返回传真记录信息
error "无权访问" 查询传真记录失败原因

说明： data取值为json数组。
json格式数据记录msg的各属性定义：
名称 描述 数据格式 取值范围 例子
enterpriseId 企业id     3000000
type 传真类型 传真类型 1:接收 2:发送   1
startTime 开始时间 时间格式“yyyy-mm-dd hh:mm:ss”   2012-12-03 15:00:02
answerTime 系统应答时间 时间格式“yyyy-mm-dd hh:mm:ss”   2012-12-03 15:00:02
endTime 结束时间 时间格式“yyyy-mm-dd hh:mm:ss”   2012-12-03 15:00:02
receiveNumber 接收方号码     01041005968
pageCount 发送/接收传真文件页数     1
file 传真文件地址      
filename 发送时有效，发送的原始文件名
