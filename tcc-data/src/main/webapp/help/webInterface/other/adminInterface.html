CCIC分平台提供给营帐内部接口

Agi Server接口

URL示例：
http://172.16.203.197/interfaceAction/adminInterface!agiServer.action?userName=ccic&password=287178f04d1be58dd1f30422933b5744&cmd=status

相关参数

参数 要求 描述 备注
userName 必选 账户 接口用户名
password 必选 密码 md5(客户密码)
cmd 必选 命令 start、stop、restart、status

返回值

JSON格式数据
{"result":"success","msg":"1"}
参数说明：
返回参数 描述
result
result返回值 描述
success 调用成功
error 参数不正确/用户验证没有通过
exception 发生异常
Msg 1表示成功,0表示失败

cmd=status返回:
JSON格式数据
{result:"success",msg:"1"}
参数说明：
返回参数 描述
result
result返回值 描述
success 调用成功
error 参数不正确/用户验证没有通过
exception 发生异常
msg 1表示agi server正在运行(对应status),0表示agi server已经关闭(对应status)
