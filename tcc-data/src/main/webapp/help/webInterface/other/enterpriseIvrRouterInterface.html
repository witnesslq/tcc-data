企业呼入路由管理接口

查询

接口URL（GET方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，随机字符串，任务类型和任务id
返回值：JSON格式返回结果
URL示例：
http://TIWebServer:TIPort/interfaceAction/enterpriseIvrRouterInterface!list.action?enterpriseId=&userName=&pwd=&seed=&start=&limit=

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业id 管理员后台登陆后，位于右上角位置
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 需和seed进行MD5加密，格式：md5(md5(对应userName的密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串
start 可选 开始 从第几条开始查询，默认为0
limit 可选 条数 查询条数，默认为10

返回值

Url示例
http://172.16.203.182/interfaceAction/enterpriseIvrRouterInterface!list.action?enterpriseId=3010066&userName=admin&seed=1234&pwd=b605e86d02eef8bfd0646f6a704c17c9br
JSON格式数据：
若操作成功{result:返回结果, code:结果代码,msg:描述,list:[{id:路由id,active:状态,priority:优先级,ruleTimeId:时间规则名称,ruleAreaName:地区规则名称,hotline:热线号码规则,routerType:路由目的类型,routerObject:路由值,description:说明,createTime:创建时间}]}
id Id编号
priority 优先级
active 1：启动 0：停用
ruleAreaId 来电地区规则，多个以;分隔
ruleTimeName 时间规则，多个以;分隔
hotline 热线号码规则，多个以;分隔
routerType 路由目的类型
routerObject 路由目的值， routerType=1，值为IVR导航ID routerType=2，值为固定号码 routerType=3，值为分机
description 路由说明
createTime 创建时间
成功返回示例：
{"result":"success","code":"0","msg":"操作成功","list":[{"id":14,"active":1,"priority":1,"ruleTimeId":"1;2","ruleAreaName":"010;020;110","hotline":"18200;4000600488","routerType":2,"routerObject":"01041005971","description":"ssss","createTime":"2014-03-26 21:41:38"},{"id":6,"active":1,"priority":3,"ruleTimeId":"","ruleAreaName":"0104100","hotline":"","routerType":1,"routerObject":"71","description":"","createTime":"2014-03-26 21:41:38"}]}

若失败{result:返回结果, code:结果代码,msg:描述}
result code msg
success 1 参数[具体参数名]不能为空
error 2 参数[具体参数名]格式不正确
error 3 企业不存在或状态不正常
error 4 无权操作，接口未开通/td>
error 5 验证失败，用户名或密码不正确
error 6 优先级已存在
error 99 其他错误，操作失败



新增/修改设置

接口URL（GET方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码
返回值：JSON格式返回结果
URL示例：
http://TIWebServer:TIPort/interfaceAction/enterpriseIvrRouterInterface!save.action?enterpriseId=?&userName=&pwd=&seed=&id=&description=&priority=&ruleAreaName=&ruleTimeId=&hotline=&routerType=&routerObject=

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业id 管理员后台登陆后，位于右上角位置
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 需和seed进行MD5加密，格式：md5(md5(对应userName的密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串
id 可选 路由id 如果是修改设置，此参数必选
ruleAreaName 可选 来电地区规则 必须为数字；支持多个，如需传入多个，传参格式如： ruleAreaName=010&ruleAreaName=020
ruleTimeId 可选 时间规则 时间条件ID，如需使用，需先配置时间条件；支持多个，如需传入多个，传参格式如：ruleTimeId=1&ruleTimeId=2
hotline 可选 热线号码规则 路由适用的热线号码；支持多个，如需传入多个，传参格式如：hotline=4006869009&hotline=4008881111
routerType 必选 路由目的类型 1：IVR导航 2：固定号码3：分机
routerObject 必选 路由目的值 routerType=1，值为IVR导航ID; routerType=2，值为固定号码; routerType=3，值为分机
priority 可选 优先级 值越小优先级越高，不能重复，默认为现有最大值+1
description 可选 路由说明

返回值

Url示例
http://172.16.203.179/interfaceAction/enterpriseIvrRouterInterface!save.action? enterpriseId=3010025&userName=admin&seed=1234&pwd=b605e86d02eef8bfd0646f6a704c17c9&routerType=1&routerObject=72
JSON格式数据：
若操作成功{result:返回结果, code:结果代码,msg:描述,id:路由id}
若失败{result:返回结果, code:结果代码,msg:描述}
返回结果说明
result code msg
success 0 操作成功
error 1 参数[具体参数名]不能为空
error 2 参数[具体参数名]格式不正确
error 3 企业不存在或状态不正常
error 4 无权操作，接口未开通/td>
error 5 验证失败，用户名或密码不正确
error 6 优先级已存在
error 99 其他错误，操作失败
