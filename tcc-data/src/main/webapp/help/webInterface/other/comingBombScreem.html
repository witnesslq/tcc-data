来电弹屏接口

API URL

接口URL（GET方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，随机字符串，任务类型和任务id
返回值：JSON格式返回结果
接口URL
URL示例：
http://www.baidu.com/?enterpriseId=3000028&hotline=4006604889&crmId=undefined&cno=2000&pwd=123456&tel=01087120766&customerNumberType=1&customerAreaCode=010&calltype=1&numberTrunk=63352451&uniqueId=xian001-1363743441.417&taskInventoryId=undefined&taskId=undefined&qid=30000280011

参数说明

参数 要求 描述 备注
enterpriseId 必选 企业id 3或4开头的数字串，开户后唯一确定
hotline 必选 企业热线号码 企业开户的热线号码，基本都是400号码，固话开户除外。
crmId 可选 坐席crm编号 crm编号
cno 必选 坐席工号 坐席工号
pwd 必选 坐席登录密码 明文密码
customerNumber 必选 主叫号码 来电呼入号码或者外呼客户号码
customerNumberType 必选 来电或外号码类型 手机或者固话
customerAreaCode 必选 号码地区 呼入电话或者外呼客户电话所在地区
calltype 必选 呼叫类型 1呼入、3外呼、2网上400等
numberTrunk 必选 目的码 目的码
uniqueId 必选 呼叫唯一标识 用于接口取得通话记录和录音等
taskInventoryId 可选 外呼任务详情ID 外呼任务的内部标识
taskId 可选 外呼任务id 外呼任务的标识
qid 可选 呼入队列号 企业编号+四位队列号

返回值

返回内容为html格式文本。
