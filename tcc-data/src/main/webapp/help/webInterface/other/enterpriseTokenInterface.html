企业获取token接口

API URL

接口URL（GET方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，加密秘钥，随机字符串
返回值：JSON格式返回结果
接口URL
URL示例：
http://InterfaceServerIp:port/interface/ivrCallback?enterpriseId=&hotline=&userName&pwd=&uniqueId=&paramNames=

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业id 管理员后台登陆后，位于右上角位置
pwd 必选 密码 需和secret、seed、客户端请求时间，进行MD5加密，格式：md5(md5(secret)+seed+yyyyMMddHHmm)。Secret是企业秘钥，yyyyMMddHHmm是客户端当前请求的时间字符串，必须精确到分钟数。例如：201404211805
seed 必选 随机字符串 例如：aaa

返回值

Url示例
172.16.203.190/interfaceAction/enterpriseTokenInterface!getToken.action?enterpriseId=3000002&pwd=d0782bdf141e366014b4bc0d7dc6ea00&seed=aaa
JSON格式数据：
{result:返回结果, code:结果代码,token:xxx,deadline:xxx}
result code msg
success 0 操作成功
error 1 参数[具体参数名]不能为空
error 2 参数[具体参数名]格式不正确
error 3 请求token超时
