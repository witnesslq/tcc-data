ivr回调接口

接口作用说明

此接口可实现ivr回调

API URL

接口URL（GET方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，随机字符串，任务类型和任务id
返回值：JSON格式返回结果
接口URL
URL示例：
http://InterfaceServerIp:port/interface/ivrCallback?enterpriseId=&hotline=&userName&pwd=&uniqueId=&paramNames=

相关参数

参数 要求 描述 备注
enterpriseId 可选 企业id enterpriseId与hotline必须二选一
hotline 可选 企业热线号码 enterpriseId与hotline必须二选一
user 必选 企业后台登录用户
pwd 必选 密码 登录密码MD5加密
uniqueId 必选 通道唯一标识
paramNames 可选 动态附带参数 参数会写入通道中，以便ivr中调用格式：以逗号分隔name1,name2…name必须为字符或数字参数名建议使用user_开头
name1 可选 动态参数 设置变量在通道中

返回值

此接口为同步接口,当通过验证开始呼叫座席后只有座席接听/呼叫超时/呼叫失败时才会返回。
JSON格式数据
{"result":"","code":" ","msg":""}
result返回值 描述
success 调用成功
error 调用失败
res返回值 描述
0 操作成功
1 参数不正确
2 账户不存在
3 用户验证没有通过
4 通道已不存在
5 回调已超时
6 其他错误
URL示例：
使用enterpriseId
http://xxx.ccic2.com/interface/ivrCallback?enterpriseId=3000072&userName=admin&pwd=66b7146abd2f6fc7d78e854f0c94cffb&uniqueId=dev18-1415787391.4&paramNames=a&a=1
成功返回：
{"result":"success","code":"0","msg":""}
失败返回：
{"result":"error","code":"5","msg":"回调已超时"}
