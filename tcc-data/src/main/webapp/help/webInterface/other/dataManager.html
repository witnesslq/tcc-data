数据存储接口

保存

接口URL（GET方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，键，值
返回值：JSON格式返回结果
接口URL
URL示例：
http://TIWebServer:TIPort/interfaceAction/dataManager!set.action?enterpriseId=&userName=&pwd=&seed=&key=&value=

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业id 管理员后台登陆后，位于右上角位置
userName 必选 企业后台登录用户 登录后台用户名，如admin
pwd 必选 密码 需和seed进行MD5加密，格式：md5(md5(对应userName的密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串
key 必选 键 不能为空
value 必选 值 不能为空；如果key已经存在，覆盖原有value

返回值

JSON格式数据
{result:返回结果, code:结果代码,msg:描述}
result code msg
success 0 操作成功
error 1 参数[具体参数名]不能为空
error 2 参数[具体参数名]格式不正确
error 3 企业不存在或状态不正常
error 4 无权操作，接口未开通/td>
error 5 验证失败，用户名或密码不正确
error 6 优先级已存在
error 99 其他错误，操作失败
