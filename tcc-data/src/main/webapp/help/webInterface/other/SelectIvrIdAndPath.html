咨询接口

API URL

接口URL（GET方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，随机字符串，任务类型和任务id
返回值：JSON格式返回结果
接口URL
URL示例：
http://ccicurl/interface/SelectIvrIdAndPath?enterpriseId=&ivrPathName=&pwd=

相关参数

参数 要求 描述 备注
enterpriseId 可选 企业id enterpriseId 例如3000001
ivrPathName 可选 Ivr节点名称
pwd 必选 密码 登录密码MD5加密

返回值

此接口为同步接口,当通过验证开始呼叫座席后只有座席接听/呼叫超时/呼叫失败时才会返回。
JSON格式数据
{"res":[]}
URL示例：

http://172.16.203.192/interface/SelectIvrIdAndPath?enterpriseId=3000004&pwd=e10adc3949ba59abbe56e057f20f883e&ivrPathName=选择
返回：
{"res":[{"企业Id":3000004,"ivr名称":"ceshi","ivrId":9,"节点":"1","节点名称":"选择"},{"企业Id":3000004,"ivr名称":"ceshi","ivrId":32,"节点":"1","节点名称":"选择"}]}
