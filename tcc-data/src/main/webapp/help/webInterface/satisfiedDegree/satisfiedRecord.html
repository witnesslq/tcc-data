满意度调查记录

接口作用说明

通过接口实现满意度调查记录的查询功能

API URL

接口URL（POST方式）
发起者：合作方
接受者：天润融通
接口类型：HTTP接口
调用方式：POST
返回格式：JSON数据
URL规范：
http://InterfaceServerIp:port/interface/entrance/OpenInterfaceEntrance?interfaceType=investigationList&enterpriseId=3000000&userName=admin&pwd=c33367701511b4f6020ec61ded352059&startDate=2013-04-23 08:00:00&endDate=2013-04-30 23:59:59
说明：InterfaceServerIp:port 为使用账号登录www.ccic2.com之后，在浏览器地址栏看到的域名地址。
例如：xa.ccic2.com
           bj.ccic2.com
           1.ccic2.com

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业Id 必选企业id
interfaceType 必选 请求类型 1.      investigationList: 获取当前企业的满意度调查记录
2.      ivrList: 获取当前企业IVR列表
3.      freeClients: 获取当前企业空闲座席
4.      webCallss
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 对应username的密码，md5(登录密码)，例如md5(123456)
cno 可选 座席工号  
startDate 必选 开始时间 开始时间不能大于结束时间
endDate 必选 结束时间 时间跨度不能大于30天
mainUniqueId 可选 通话记录唯一标识 例如：ccic_dev_11-1400747750.13
keys 可选 评分值 例如：1
callType 可选 呼叫类型 1.呼入  2.网上400 3.点击外呼 4.预览外呼 5.预测外呼6.直接外呼 7.自动录音 8.预约回呼
