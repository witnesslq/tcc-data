满意度调查设置接口

接口作用说明

通过接口实现满意度调查设置功能

API URL

接口URL（POST方式）
发起者：合作方
接受者：天润融通
接口类型：HTTP接口
调用方式：POST
返回格式：JSON格式数据
URL规范：
http://InterfaceServerIp:port/interfaceAction/investigationInterface!save.action?enterpriseId=&userName=&pwd=&ivr=&auto=
说明：InterfaceServerIp:port 为使用账号登录www.ccic2.com之后，在浏览器地址栏看到的域名地址。
例如：xa.ccic2.com
           bj.ccic2.com
           1.ccic2.com

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业Id  
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 对应username的密码，md5(登录密码)，例如：md5(123456)
ivr 必选 满意度调查ivr流程 json格式字符串，具体参考ivr格式说明
传此参数ivrId参数失效
需要将json进行urlencode编码
auto 可选 是否自动满意度调查 1：打开自动满意度调查
0：手动满意度调查
如果不传此参数默认为手动满意度调查

IVR参数格式说明

ivr参数json格式，所有值均为字符串格式
{"nodes": [
node,
node
…
]
}
参数 要求 描述 备注
nodes 必选 节点列表 json数组，必须有1个或以上节点
node json格式
{
            "nodeId": "1",
            "nodeName": "播放",
            "action": "1",
                     "${param}": "1",
                     "${param}": "1",
…
}
参数 要求 描述 备注
nodeId 必选 节点ID 字符串(仅包含数字及点号)，例如1，1.1，1.1.2，
节点以点号区分层级关系，例如1.1.2的父节点是1.1
节点根节点必须是1
ivrName 必选 节点名称 字符串
action 必选 节点类型 参考action取值说明
${param} 必选 其他参数 随action不同，有不同的参数要求，具体参考下述节点参数说明表
action取值说明：
值 节点类型
1 播放节点
2 选择节点
 
节点参数说明表：
action=1播放节点
参数 要求 描述 备注
playType 必选 播放类型说明 参考playType取值说明表
playFile 必选 播放的文件名或内容 当playType=1时为播放的文件名
当playType=2时为数字值
当playType=3时为数字值
当playType=4时为用于TTS的文本值
next 必选 跳转节点 如果跳转节点
action=2选择节点
参数 要求 描述 备注
selectFile 必选 播放语音文件名字  
select${x}Next 可选 记录不同按键将会跳转的节点；${x}的值可以是0~9,*以及#； 不同按键将要跳转的节点；如select1Next=1.2表示在选择节点按了数字1键，对应的跳转到节点1.2；
select4Next=1.3表示客户在选择节点按了数字4键，对应的跳转节点是1.3；如果某按键的跳转节点为空将会导致错误按键，跳转到错误按键的节点；
 
playType取值说明表:
值 节点类型
1 语音文件
2 数字
3 数目
4 TTS文本

返回值

此接口返回JSON格式数据。
{“result”:”success”,”description”:””}
result列表
result description
success 提交成功
error 提交失败
description为失败时错误描述
 
每次提交会覆盖之前提交的设置。
示例
http://172.16.203.217/interfaceAction/investigationInterface!save.action?enterpriseId=3000000&userName=admin&pwd=e10adc3949ba59abbe56e057f20f883e&auto=1&ivr={"nodes":[{"nodeId":"1","ivrName":"选择","action":"2","selectType":"1","selectFile":"on_duty","select1Next":"1.1","select2Next":"1.2","select3Next":"1.3"},{"nodeId":"1.1","ivrName":"播放1","action":"1","playType":"1","playFile":"off_duty","next":""},{"nodeId":"1.2","ivrName":"播放2","action":"1","playType":"1","playFile":"default_voice_mail","next":""},{"nodeId":"1.3","ivrName":"播放3","action":"1","playType":"1","playFile":"default_transfer","next":""},{"nodeId":"1.4","ivrName":"播放4","action":"1","playType":"1","playFile":"investigation_bye","next":""}]}
