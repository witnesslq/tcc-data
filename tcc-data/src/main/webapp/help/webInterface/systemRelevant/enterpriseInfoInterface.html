热线号码查询企业编号接口

接口作用说明

此接口可实现通过热线号码查询企业编号功能

API URL

接口URL（POST方式）
发起者：合作方
接受者：天润融通
接口类型：HTTP接口
调用方式：POST
返回格式：JSON数据
URL规范：
http://InterfaceServerIp:port/interfaceAction/enterpriseInfoInterface!getEnterpriseId.action

相关参数

参数 要求 描述 备注
hotline 可选 企业热线号码 热线号码，如：4006007777
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 对应username的密码，md5(md5(登录密码))，例如：md5(md5(123456))

返回值

URL实例：
http://172.16.203.190/interfaceAction/enterpriseInfoInterface!getEnterpriseId.action?hotline=4006007777&userName=admin&pwd=e10adc3949ba59abbe56e057f20f883e
 
此接口返回JSON格式数据。
JSON格式数据
{"result":"success","msg":"enterpriseId:3000005"}
