企业信息接口

接口作用说明

查看企业信息接口。
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：呼叫中心系统IP
接口参数：企业账号、企业账户密码
接口返回值：操作结果
接口URL
URL示例：
http://172.16.203.190/interfaceAction/enterpriseInfoInterface!view.action?enterpriseId=3000002&userName=admin&password=e10adc3949ba59abbe56e057f20f883e
参数说明
该示例中password = e10adc3949ba59abbe56e057f20f883e = MD5Encoder.encode('123456')
参数 要求 描述 备注
enterpriseId 必选 企业id 例如3000002
userName 必选 用户名 登录后台用户名，如admin
password 必选 登录密码 md5(登录密码)
返回值
JSON串，格式为{result:xxx,msg:xxx},result表示返回结果状态码，msg表示结果状态的描述。 {"result":"success","msg":{"企业ID：":3000002,"企业名称：":"wangjing","企业全 称：":"wangjing","所属区号：":"010","负责人姓名：":"wangjing","称谓：":"先生","移动电 话：":"15120054657","固定电话：":"","电子信箱：":"wangjing301@ti-net.com.cn","传 真：":"","通信地址：":"","邮政编码：":"","法人：":"wangjing","所属行业：":"","营业执照号码：":"","国 家：":"China","省份：":"","城市：":"","创建时间：":"2013-07-23 14:17:17.929389"}}


result与msg属性值的定义：
result msg
success 查询成功
error 失败原因
