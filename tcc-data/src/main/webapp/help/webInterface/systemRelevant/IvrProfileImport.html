企业IVR管理

接口作用说明

通过接口实现对企业IVR的管理

API URL

接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码
接口返回值：JSON格式返回结果
接口URL
URL示例：
http://TIWebServer:TIPort/interface/ivr/IvrProfileImport?enterpriseId=&userName=&pwd=&seed=&action=&ivrProfileId=&stringIvr=

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业编号 管理员后台登陆后，位于右上角位置
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 需和seed进行MD5加密，格式：md5(md5(对应userName的密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串  
action 必选 操作 取值为：
add：新增IVR
delete：删除IVR
update：更新IVR
ivrProfileId 必选 IVR ID action=delete和action=update时，此参数必选，表示要删除或更新的IVR导航的ID
stringIvr 必选 IVR导航内容 参数需要URLEncode；action=add和action=update时，此参数必选，表示要新增和更新的IVR导航内容；

JSON格式数据
action=add时，如果导入成功，将返回ivrProfileId，
{result:返回结果, code:结果代码,msg:描述,ivrProfileId:’’}
其他操作时返回：
{result:返回结果, code:结果代码,msg:描述}
 
result code msg
success 0 导入、更新或者删除成功
error 1 参数[具体参数名]不能为空
error 2 参数[具体参数名]格式不正确
error 3 传入的企业[[具体参数名]有误
error 4 无权操作，接口未开通
error 5 密码错误
error 6 导入ivr出现异常，请联系管理员
