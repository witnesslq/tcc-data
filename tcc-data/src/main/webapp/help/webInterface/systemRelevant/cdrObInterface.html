查看外呼任务接口

接口作用说明

此接口可实现查看外呼任务状态

API URL

接口URL（POST方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，随机字符串，任务类型和任务id
返回值：JSON格式返回结果
接口URL
URL示例：
http://172.16.203.197/interfaceAction/cdrObInterface!getTasks.action?enterpriseId=&userName=admin&pwd=&seed=&type=&id=

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业id 例如:3000008
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 需和seed进行MD5加密，格式：md5(md5(对应userName的密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串 随机字符串：例如：aaa
type 可选 任务类型 1表示预览外呼,2表示预测外呼,3表示IVR
id 可选 任务id 根据任务的id查询该条外呼任务详情

返回值

URL实例：
http://172.16.203.190/interfaceAction/cdrObInterface!getTasks.action?enterpriseId=3000002&userName=admin&pwd=d0782bdf141e366014b4bc0d7dc6ea00&seed=aaa&type=1&id=1
 
根据参数returnType返回相应格式的数据，若该参数值不对，返回default的JSON格式的提示信息。
返回JSON：
JSON串，格式为
{result:返回结果,msg:[{来电IVR详细记录数据},{}..]}
result、 msg与description定义：
result msg description count
success {"result":"success","msg":[{"id":"1","name":"1231231","statu":"未执行"}]} 成功返回外呼任务 实际返回数据记录数
说明： data取值为json数组。
json格式数据记录msg的各属性定义：
名称 描述 数据格式 取值范围 例子
id 任务id     1
name 任务名称     任务1
statu 任务状态     执行中...
