隐藏号码解密

接口作用说明

在开启号码隐藏功能后，可通过此接口实现号码的解密

API URL

接口URL（POST方式）
接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，用户名，密码，随机字符串，加密的客户电话号码
返回值：JSON格式返回结果
接口URL
URL示例：
http://TIWebServer:TIPort/interfaceAction/customerInterface!decode.action?enterpriseId=&userName=&pwd=&seed=&customerNumber=

相关参数

参数 要求 描述 备注
enterpriseId 必选 企业编号 管理员后台登陆后，位于右上角位置
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 需和seed进行MD5加密，格式：md5(md5(对应userName的密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串  
customerNumber 必选 被加密的电话号码 例如：010BYBE5932

返回值

URL实例：
http://172.16.203.190/interfaceAction/customerInterface!decode.action?enterpriseId=3000002&userName=admin&pwd=d0782bdf141e366014b4bc0d7dc6ea00&seed=aaa&customerNumber=010BYBE5932
JSON格式数据：
{result:返回结果, code:结果代码,msg:[customerNumber:xxx]}
result code msg
success 0 操作成功返回信息
error 1 参数[具体参数名]不能为空
error 2 用户名密码不正确
error 3 customerNumber解密失败
