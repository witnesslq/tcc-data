黑白名单

接口作用说明

企业用来配置或者新增黑、白名单

设置黑白名单

接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，请求类型，用户名，密码接口返回值：JSON格式返回结果
接口URL
URL示例：
http://TIWebServer:TIPort/interface/entrance/OpenInterfaceEntrance?interfaceType=setRestrictTel&enterpriseId=&userName=&pwd=&setRestrictTelType=
参数说明
参数 要求 描述 备注
enterpriseId 必选 企业编号 管理员后台登陆后，位于右上角位置
interfaceType 必选 请求类型 interfaceType=setRestrictTel   设置黑白名单
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 需进行MD5加密，格式：md5(对应userName的密码)，例如：md5(123456)
setRestrictTelType 必选 设置黑白名单开启状态 0表示关闭黑白名单，1表示开启黑名单，2表示开启白名单
 
返回值
URL实例：
http://172.16.203.190/interface/entrance/OpenInterfaceEntrance?interfaceType=setRestrictTel&enterpriseId=3000002&userName=admin&pwd=e10adc3949ba59abbe56e057f20f883e&setRestrictTelType=0 
JSON格式数据：
{result:返回结果, code:结果代码,msg:描述}
result code msg
success 0 操作成功返回信息
error 1 参数[具体参数名]不能为空
error 2 参数[具体参数名]不正确

管理黑名单

接口类型：HTTP请求
接口使用账户：企业账户
接口地址：天润呼叫中心接口服务器IP或域名
接口参数：企业编号，请求类型，用户名，密码，客户号码，操作代号
接口返回值：JSON格式返回结果
接口URL
URL示例：
http://TIWebServer:TIPort/interface/entrance/OpenInterfaceEntrance?interfaceType=blacklist&enterpriseId=&userName=&pwd=&customerNumber=&action=
参数说明
参数 要求 描述 备注
enterpriseId 必选 企业编号 管理员后台登陆后，位于右上角位置
interfaceType 必选 请求类型 interfaceType=blacklist   黑名单
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 需进行MD5加密，格式：md5(对应userName的密码)，例如：md5(123456)
customerNumber 必选/可选 客户号码 只能是座机或者手机号码；
action=1时可选
action 可选 操作代号 默认是0表示新增，1表示查询，2表示删除
 
返回值
URL实例：
新增：
http://172.16.203.190/interface/entrance/OpenInterfaceEntrance?interfaceType=blacklist&enterpriseId=3000002&userName=admin&pwd=e10adc3949ba59abbe56e057f20f883e&customerNumber=15120054657&action=0 
查询：
http://172.16.203.190/interface/entrance/OpenInterfaceEntrance?interfaceType=blacklist&enterpriseId=3000002&userName=admin&pwd=e10adc3949ba59abbe56e057f20f883e&customerNumber=15120054657&action=1
删除：
http://172.16.203.190/interface/entrance/OpenInterfaceEntrance?interfaceType=blacklist&enterpriseId=3000002&userName=admin&pwd=e10adc3949ba59abbe56e057f20f883e&customerNumber=15120054657&action=2
JSON格式数据：
action=0   {result:返回结果, description:描述}
action=1   {result:返回结果, code:结果代码,msg:[[“enterpriseId”:”xxx”, “tel”:“xxx”, “comment”:”xxx”}]
action=2   {result:返回结果, code:结果代码,msg:描述]
 
result msg
success ok
error 黑名单号码已经存在，不能重复添加
error 用户名错误
error 企业ID错误
error 用户名密码不正确
