来电通话记录DB同步

实现步骤

接口URL（POST方式）
发起者：合作方
接受者：天润融通
接口类型：HTTP接口
调用方式：POST
返回格式：JSON数据
URL规范：
http://InterfaceServerIp:port/interfaceAction/cdrIbInterface!cdrIbDBCopy.action
说明：InterfaceServerIp:port 为使用账号登录www.ccic2.com之后，在浏览器地址栏看到的域名地址。
例如：xa.ccic2.com
           bj.ccic2.com
           1.ccic2.com

相关参数

参数 要求 描述 备注
hotline 可选 企业热线号码 与enterpriseId二选一
enterpriseId 可选 企业Id 与hotline二选一
userName 必选 用户名 登录后台用户名，如admin
pwd 必选 密码 对应username的密码，md5(md5(登录密码)+seed)，例如：md5(md5(123456)seed)
seed 必选 随机字符串  
id 必选 数据表主键标识 正整数，大于等于0，默认0
limit 可选 通话记录条数 正整数，大于等于1，默认10，最大不能超过5000
注：每次请求返回行数最大值为limit（最大不能超过5000条），即从id大于传入的参数id的值开始按id升序取出limit条记录。

返回值

返回值
此接口返回JSON格式数据。
JSON格式数据
{result:返回结果,
msg:{
data:[{通话记录数据},{}..],
totalCount:总记录数
}
}
